# SLA DE SERVIÇOS — LUMMÉ

**SLA = Service Level Agreement (Acordo de Nível de Serviço)**

Este documento define os prazos, padrões e condições de entrega para cada serviço da Lummé.

---

## PRINCÍPIO DO SLA Lummé

> Prazo combinado é prazo cumprido. O único jeito de construir confiança com clientes é entregando o que prometemos, quando prometemos.

## PADRÕES DE QUALIDADE

### Checklist de qualidade antes de enviar qualquer entrega ao cliente

**Para peças visuais:**
- [ ] Ortografia revisada (Grammarly/revisão manual)
- [ ] Paleta de cores do cliente respeitada
- [ ] Tipografia aprovada no briefing
- [ ] Hierarquia visual clara (o olho sabe para onde ir)
- [ ] Versão mobile testada
- [ ] Tamanhos corretos para cada plataforma

**Para copy:**
- [ ] Resultado apresentado antes do processo
- [ ] Voz ativa predominante
- [ ] Termos técnicos explicados na mesma frase
- [ ] Tom de voz do cliente preservado (conforme briefing)
- [ ] CTA claro e único por peça
- [ ] Máximo 25 palavras por frase (exceções justificadas)

**Para relatórios:**
- [ ] Dados precisos — cruzados com a fonte original
- [ ] Análise qualitativa além dos números
- [ ] Insights acionáveis (não só dados, mas o que fazer com eles)
- [ ] Próximos passos claramente definidos

## POLÍTICA DE REVISÕES

| Tipo de serviço | Revisões gratuitas | Revisão adicional |
|-----------------|---------------------|------------------|
| Posts de feed | 2 por peça | R$ 80/rodada |
| Roteiro de Reel | 2 por roteiro | R$ 120/rodada |
| Identidade visual | 2 por etapa | R$ 200/hora |
| Diagnóstico | 1 pós-apresentação | R$ 200/hora |
| Proposta comercial | 1 | R$ 150/hora |

**Revisão válida:** Ajustes no escopo combinado, correção de erros de interpretação do briefing, melhorias na copy ou visual dentro do tom acordado.

**Revisão NÃO válida:** Mudança completa de direção, alteração de identidade visual já aprovada em etapa anterior, pedidos fora do escopo contratado.

---

## COMUNICAÇÃO EM CASO DE ATRASO

Se a Lummé identificar risco de atraso em qualquer entrega:

1. Comunicar o cliente com **antecedência mínima de 48h**
2. Apresentar nova data de entrega
3. Explicar o motivo brevemente
4. Propor compensação se o atraso for significativo (ex: crédito em serviço)

**O que nunca fazer:** Deixar o cliente descobrir o atraso por conta própria no dia da entrega.

---

## ESCALONAMENTO DE PROBLEMAS

| Situação | Ação |
|----------|------|
| Cliente insatisfeito com entrega | Reunião de alinhamento + nova proposta de solução em 24h |
| Cliente não aprova após 2 revisões | Reunião para entender o gap + avaliar se é escopo ou expectativa |
| Conflito sobre escopo | Retornar ao contrato e proposta original |
| Atraso grave (>5 dias) | Crédito proporcional na próxima fatura |
| Cancelamento do cliente | Seguir processo de offboarding documentado |
