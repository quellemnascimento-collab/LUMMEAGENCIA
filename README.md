## O que é a Lummé

A Lummé não é uma agência de marketing convencional. É uma **casa criativa premium** que ilumina marcas com inteligência, beleza e direção estratégica — para que a presença digital reflita o valor real do que você entrega.

> *"Não é sobre aparecer mais. É sobre aparecer com a presença que o seu trabalho merece."*

### Os 4 Layers da Metodologia Lummé

| Layer | O que entrega |
|-------|--------------|
| **1. Estrutura Visual** | Design com hierarquia e identidade calibrada para conversão |
| **2. Copy Estratégica** | Texto com lógica de conversão e variáveis de nicho |
| **3. Prompt de IA** | Instrução pronta para personalizar em 30 segundos |
| **4. Instrução de Veiculação** | Rede, horário e formato ideais para cada post |
